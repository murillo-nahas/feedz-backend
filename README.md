# feedz
A social media API to explore MongoDB modeling patterns, built with NestJS, Mongoose and BullMQ.

## Stack
| Layer    | Tech                    |
| -------- | ----------------------- |

| Database | MongoDB Atlas           |
| ODM      | Mongoose                |
| Queue    | BullMQ + Redis          |
| Auth     | JWT                     |
